# SkillCraft Technology Internship - Task 4

## Project Title
Marketing Campaign Analysis

## Description
This project focuses on analyzing a marketing campaign dataset using Python. The data is cleaned, explored, and visualized to identify customer behavior, campaign performance, and business insights.

## Files Included
- Marketing Campaign Analysis Report_T4.docx – Project report
- Task 4_T4.ipynb – Jupyter Notebook containing the complete analysis
- marketing_campaign_T4.csv – Dataset used for analysis

## Tools & Technologies
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Objectives
- Load and preprocess the marketing campaign dataset
- Perform exploratory data analysis (EDA)
- Visualize important trends and patterns
- Generate meaningful business insights

## Outcome
The project provides valuable insights into customer behavior and marketing campaign effectiveness through data analysis and visualization.
